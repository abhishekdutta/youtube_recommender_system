How to run this:

  1. Download the zip file by click button above
  2. Issue the command cd to your code path
  3. Issue the command python -m SimpleHTTPServer to start the server
  4. Open your browser http://localhost:8000/index.html to visualize data!

Known issues

  1. since d3.js uses ajax to load tsv files, the files contained in the src folder cannot be run in a browser directly using file://index.html. Please execute the above commands under the src directory to create a HTTP server. Otherwise it will display NOTHING!
